# 简介

- 1732[.3.31](2024-03-31.md)~1809[.5.31](2024-05-31.md)，维也纳古典乐派奠基人。
- 多产的作曲家， “交响曲之父”、“弦乐四重奏之父”、 “海顿爸爸“。
- 与妻子40年婚姻关系十分不和睦，无子，并经常出轨。
- 作品明快健康，旋律清新明快，结构合理于民间音乐关系紧密，区别于同时代其他作曲家，极具个人特色。
- [莫扎特](莫扎特.md)和[贝多芬](贝多芬.md)的老师。
# 主要作品

《第45号交响曲》、《第88号交响曲》。
