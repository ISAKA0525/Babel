# 简介

- 1906[.10.20](2024-10-20.md)~1955[.2.17](2024-02-17.md)，新潟县新潟市。本名坂口炳五（生于丙午年，家中排行第五）。
- 东洋大学[文学部](文学部.md)印度哲学科毕业。
- 无赖派。
- 第二部侦探作品《复员杀人事件》因杂志《座谈》倒闭而被迫中断，后由[高木彬光](高木彬光.md)加笔完成。
- “余将成为伟大的落伍者，有朝一日重现于历史之上。“
